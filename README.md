# SetPremissionsFromExcel
This program contains to separate scripts , "XlsxToCsv.ps1" will export your xlsx file to csv and "ExcelToPowershell.ps1" will set your desired NTFS premissions from your excel file , it mean you only need change premissions in your excel and you doesnt need to manually set premissions in NTFS premissions , R ( READ) , W (Write) , M (Modify)
